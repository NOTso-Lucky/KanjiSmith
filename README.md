## Overview

KanjiSmith is a full-stack Japanese learning platform designed to make vocabulary acquisition efficient through intelligent flashcards and a personalized Spaced Repetition System (SRS).

The project aims to provide a complete learning ecosystem consisting of:

* 🌐 Web Application
* 🧩 Browser Extension
* 📱 Android Application
* ☁️ Shared Cloud Backend

The long-term goal is to create a centralized platform where users can discover, organize, review, and master Japanese vocabulary across multiple devices.
