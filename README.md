# KanjiSmith 🇯🇵

> A modern Japanese learning platform built with **FastAPI** and **React**.

KanjiSmith is designed to make learning Japanese vocabulary intuitive through intelligent flashcards, detailed dictionary lookups, and a seamless study experience. The long-term vision includes a browser extension for one-click word saving, spaced repetition, and personalized learning.

---

## ✨ Features

### Current
- 🔐 JWT Authentication (In Progress)
- 👤 User Registration & Login
- 🔒 Secure password hashing with bcrypt
- 📚 Japanese dictionary lookup
- 🔤 Romaji conversion
- 📝 Example sentences
- ⚡ FastAPI REST API
- ⚛️ React + Vite frontend

### Planned
- 🎴 Flashcard decks
- 🧠 Spaced Repetition System (SRS)
- 📈 Learning progress dashboard
- 🌐 Browser extension
- ⭐ Favorite words
- 🔍 Advanced search & filters
- 🎧 Audio pronunciation
- 📱 Responsive UI
